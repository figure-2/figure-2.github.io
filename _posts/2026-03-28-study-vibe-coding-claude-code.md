## 추가 정리

### 핵심 요약

이 글은 바이브코딩과 Claude Code를 처음 교육할 때 사용할 자료 묶음으로 보면 된다. 핵심은 도구 사용법 자체보다, 자연어 요청을 작업 단위로 쪼개고 결과물을 검토하는 습관을 만드는 것이다.

### 보충 해설
