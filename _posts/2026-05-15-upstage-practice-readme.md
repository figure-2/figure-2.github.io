## Exercises - 자기주도 실습

이 폴더에는 각 주차별 실습 과제가 포함되어 있습니다.
Claude Code에서 `/mission` 명령으로 미션 도전 후, 여기서 추가 실습을 진행하세요.

## 실습 목록

### Week02 - 자료구조와 알고리즘
- `w02-stack-queue.py` - 스택/큐 구현 연습

### Week05 - Prompt Engineering
- `w05-prompt-design.md` - 프롬프트 설계 연습

### Week08 - Agent 아키텍처
- `w08-langgraph-basic.py` - LangGraph 기본 그래프 구현

## 실습 방법

1. 각 파일을 열어 작성 지시 주석을 확인하세요
2. 코드를 완성하고 실행해보세요
3. Claude Code에서 `/ask` 명령으로 도움을 받을 수 있습니다

## 정리 기준

- 실습 파일을 단순 목록으로만 보지 않고, 어떤 주차의 어떤 개념을 확인하기 위한 과제인지 함께 기록한다.
- 코드 실습은 실행 결과, 막힌 지점, 다시 확인한 개념을 같이 남긴다.
- 프롬프트 실습은 입력 프롬프트, 모델 응답, 개선한 프롬프트를 비교해서 정리한다.

## 체크포인트

- [ ] 각 실습이 연결되는 수업 주차를 확인했다.
- [ ] 실행 결과 또는 작성한 답안을 남겼다.
- [ ] 다시 볼 개념을 관련 글과 연결했다.
